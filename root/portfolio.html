SAT Mistake Inputter

Technologies Used

HTML
CSS
Javascript
Python

Description

This is a tool I made for an SAT bootcamp to help students record mistakes that they made. After reviewing missed questions, they color it green and the question goes to a different column.

The script organizes cells and adds new questions. Based on the number of questions missed in a section the percent likelihood of getting a certain score is presented.
The sidebar was created using HTML and CSS with the logic being executed with Javascript. Python was used to data scrape the curves for the past 13 SATs for each section from the College Board in order to give the students an accurate score based on their performance.
